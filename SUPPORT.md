# 支持说明

## 使用问题

开始前请查阅：

- [README](README.md)
- [环境变量与运行配置](docs/configuration.md)
- [冒烟测试](docs/smoke-testing.md)
- [图片任务排障](docs/image-task-observability.md)

如果确认是可复现缺陷，可使用仓库的 Bug Report 模板创建 Issue。请提供运行模式、Node.js 版本、复现步骤、预期结果、实际结果和脱敏日志。

## 功能建议

使用 Feature Request 模板说明用户问题、预期价值、建议范围和兼容要求。提交建议不代表功能已进入 Roadmap 或承诺发布日期。

## 不适合公开讨论的内容

不要在 Issue 中提交 API Key、Webhook、数据库地址和密码、证书、手机号、邮箱、订单、用户图片或支付通知原文。安全漏洞请遵循 [SECURITY.md](SECURITY.md)。

本仓库暂不承诺商业 SLA 或一对一部署支持。
