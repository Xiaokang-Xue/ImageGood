# 社区行为准则

ImageGood 希望为所有参与者提供尊重、专业且安全的协作环境。

## 期望行为

- 围绕技术事实和用户问题展开讨论，尊重不同经验与观点。
- 给出可复现的信息，接受合理的代码审查和范围调整。
- 对安全、隐私、支付和用户数据保持谨慎。
- 在发生分歧时关注改动本身，不进行人身评价。

## 不可接受的行为

- 骚扰、歧视、侮辱、威胁或公开他人的私人信息。
- 在 Issue、PR、日志或截图中披露密钥、证书、个人信息或未修复漏洞。
- 恶意刷屏、破坏协作流程，或在被要求停止后继续不当行为。
- 冒充维护者、用户或第三方服务提供方。

## 处理方式

维护者可以编辑或删除不当内容、关闭讨论、拒绝贡献，或在严重情况下限制参与。处理时应尽量说明适用规则，并保护报告者隐私。

行为问题请通过仓库所有者提供的非公开联系方式报告。安全漏洞请遵循 [SECURITY.md](SECURITY.md)，不要在公开 Issue 中披露细节。

本准则适用于仓库、Issue、Pull Request、代码审查及代表项目参与的其他公开空间。
