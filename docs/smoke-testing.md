# 核心功能自动化冒烟测试

冒烟测试用于在部署前后快速确认 ImageGood 的核心页面、公开接口和权限边界是否正常。测试不会提交图片生成任务、不会扣除积分，也不会创建支付订单。

## 本地执行

先启动网站：

```bash
npm run dev -- -H 0.0.0.0
```

再打开另一个终端执行：

```bash
npm run test:smoke
```

默认检查 `http://127.0.0.1:3000`。

## 检查线上环境

```bash
npm run test:smoke -- --base-url=https://imagegood.net
```

也可以通过环境变量指定地址：

```bash
SMOKE_BASE_URL=https://imagegood.net npm run test:smoke
```

## 检查范围

- 首页、登录、注册、价格页。
- AI 修图、文生图、智能抠图、图片增强、去杂物、商品图、封面海报。
- 算术验证码和积分套餐公开接口。
- 当前用户、历史记录、管理员看板的未登录访问保护。

权限接口在未登录状态下返回 `401` 表示保护正常，因此会显示为 `PASS`。任一检查失败时，命令以非零状态退出，适合部署脚本和持续集成使用。

如开发环境首次编译页面较慢，可提高单项超时时间：

```bash
npm run test:smoke -- --timeout-ms=60000
```
